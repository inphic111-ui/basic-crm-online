## 技術細節

### 數據庫架構
- 客戶表：customers
- 訂單表：orders（待建立）
- 音檔表：audio_files（待建立）

### 後端 API
- GET /api/customers - 獲取客戶列表（支援搜尋和篩選）
- GET /api/customers/:id - 獲取客戶詳情
- PUT /api/customers/:id - 更新客戶信息
- DELETE /api/customers/:id - 刪除客戶
- POST /api/audio - 上傳音檔
- POST /api/ai/analyze - AI 分析音檔

### 前端頁面
- 客戶清單（支援搜尋和篩選）
- 客戶詳情彈窗（顯示購買歷史、音檔、AI 分析）
- 錄音管理頁面（待建立）
- 訂單列表頁面（待建立）
